# Basic Health Screening Tool

A Shiny app for basic health screening with **BMI** and **blood pressure assessment**.  
This tool collects simple demographic and anthropometric data, calculates BMI, interprets blood pressure, and provides an overall risk assessment with recommended actions.

### 🖼 Screenshot
![Basic Health Screening Tool Screenshot](app_image.png)

---

## 🚀 Features
- Input fields for age, sex, weight, height, and blood pressure
- Automatic BMI calculation and classification
- Blood pressure interpretation (normal, elevated, hypertension stages)
- Overall risk assessment and recommended actions
- Simple, user‑friendly interface built with R Shiny
